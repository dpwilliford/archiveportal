# Sarasota LGBT Archive Portal

The Archive Portal is the public access layer of a three-part archival software system:

1. Intake Workbench (offline-first field capture and SIP creation)
2. Archive OS (preservation, metadata, rights, and finding-aid system of record)
3. Archive Portal (public publication of approved DIPs only)

## Core Principle

Field app collects and packages. Archive OS preserves and governs. Archive Portal publishes only approved public derivatives and metadata.

See the `docs/` directory for architecture, metadata standards, finding-aid requirements, data boundaries, and the development roadmap.
