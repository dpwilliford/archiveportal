# Content Model

## Core Public Entities

- Collection
- Series
- Subseries
- File
- Item
- DigitalObject
- OralHistory
- Transcript
- Person
- Organization
- Subject
- Place
- Exhibit
- TimelineEvent
- Publication
- RightsStatement

## Hierarchical Description

Archival description follows a hierarchy:

Collection → Series → Subseries → File → Item

Digital objects may be attached at any descriptive level.

## Authority Relationships

People, organizations, subjects, and places may be linked to any descriptive record, oral history, publication, or exhibit.

## Publication Constraint

Only records in an approved public export may appear in the Portal.
