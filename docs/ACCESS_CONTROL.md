# Access Control

## Principle

Access control must protect privacy, agency, donor intent, and community safety. Public visibility must never be assumed.

## Roles

Initial roles:

- public visitor
- registered community member
- verified researcher
- donor
- volunteer
- contributor
- editor
- board member
- archivist_owner
- administrator

## Permission domains

Permissions must control:

- reading content
- reading restricted metadata
- seeing thumbnails
- viewing full media
- downloading media
- submitting forms
- editing drafts
- publishing
- withdrawing
- viewing dashboards
- viewing takedown requests
- managing users
- managing integrations

## Explicit material rules

Sexually explicit material must:

- never be public by default
- require login
- require age affirmation
- suppress public thumbnails unless explicitly approved
- disable public social sharing
- disable search-engine indexing
- require access logging
- require archivist approval before publication
