# Data Boundaries and Publication Rules

## Permitted in Portal

- Approved public metadata
- Access derivatives
- Public transcripts
- Public rights statements
- EAD XML
- PDF finding aids
- IIIF manifests

## Prohibited in Portal

- Preservation masters
- Donor contact information
- Private consent forms
- Restricted files and metadata

## Publication Workflow

draft → in_review → approved → published → withdrawn → superseded
