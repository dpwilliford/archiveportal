# Architecture

```text
Intake Workbench → Archive OS → Archive Portal
```

## Information Packages

- SIP (Submission Information Package) — created by Intake Workbench
- AIP (Archival Information Package) — governed by Archive OS
- DIP (Dissemination Information Package) — exported by Archive OS to the Portal
