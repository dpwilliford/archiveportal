# MVP Scope

## Objective

Launch a credible public website for the Sarasota LGBT Archive as quickly as possible.

The MVP establishes:

- institutional legitimacy
- a stable public URL
- mission and governance information
- donor and volunteer outreach
- a historical timeline
- initial publications
- a foundation for grants and fundraising

## Success Criteria

The MVP is successful when:

1. The site is publicly accessible on the production domain.
2. All core informational pages are complete.
3. The site is mobile responsive.
4. Basic accessibility and metadata are in place.
5. Contact and newsletter pathways function.
6. The site can be updated through code changes and redeployment.

## Required Pages

- Home
- About
- Mission
- Timeline
- Publications
- Oral Histories (coming soon)
- Get Involved
- Donate
- Contact
- Privacy Policy

## Content Storage

Initial content should be stored as Markdown or TypeScript data files in the repository.

No database is required for the MVP.

## Out of Scope

- PostgreSQL
- Prisma models
- Authentication
- Admin CMS
- Media upload
- User accounts
- Search and browse
- Restricted records
- Dashboards
- Public submission workflows
- Archive OS synchronization

## Deployment Goal

The MVP should be deployable within one week of focused development.
