# Media Specification

## Principle

Images and videos are archival media records, not decorative assets.

## Image metadata

Every public image must include:

- title
- caption
- altText
- creator
- photographer
- dateDisplay
- dateStart
- dateEnd
- placeId
- sourceCollection
- rightsStatement
- creditLine
- ArchiveOSIdentifier
- originalFilename
- derivativeFilenames
- visibilityLevel
- consentStatus
- sensitivityLevel
- containsIdentifiablePeople
- relatedPeople
- relatedPlaces
- relatedEvents
- relatedExhibits

## Rules

- Never serve preservation masters.
- Captions must always be visible or directly accessible.
- Metadata must not depend on hover states.
- Mobile display must prioritize visible captions and expandable metadata.
- Documentary and object images must not be cropped by default.
- Sensitive or explicit material must suppress public thumbnails unless approved.
