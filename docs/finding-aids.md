# Finding Aids

Archive OS generates and versions finding aids; the Portal renders the approved public version.

## Required Sections

- Title statement
- Repository
- Creator
- Collection identifier
- Dates
- Extent
- Abstract
- Biographical/Historical Note
- Scope and Content
- Arrangement
- Access and Use Restrictions
- Preferred Citation
- Related Materials
- Controlled Access Headings
- Container List
- Digital Objects
- Revision History
