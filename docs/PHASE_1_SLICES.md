# Phase 1 Slices: Barebones Public Site

## Goal

Launch a functioning public website for the Sarasota LGBT Archive as quickly as possible while preserving the architectural boundaries required for later phases.

Every slice must:

- produce a visible, testable result
- preserve all existing functionality
- keep the application deployable
- include verification steps
- defer unrelated work explicitly
