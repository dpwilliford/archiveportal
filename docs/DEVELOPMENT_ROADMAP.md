# Development Roadmap

## Method

Build thin vertical slices. Each slice must produce a visible, testable feature.

## Slice 0: Repository and documentation

- create repository
- add foundational docs
- add .env.example
- add baseline README
- add issue/PR workflow conventions

## Slice 1: Public shell

- homepage
- about page
- navigation
- footer
- responsive layout
- placeholder routes

## Slice 2: Admin login and roles

- archivist login
- protected /admin
- basic roles: public, archivist, administrator

## Slice 3: Pages CMS

- create/edit/publish pages
- title, slug, body, status
- draft/published state

## Slice 4: Media records

- image upload
- caption required
- alt text required
- rights statement required
- image detail page
