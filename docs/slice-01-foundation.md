# Slice 01: Public Foundation

## Objective

Create a deployable public website with core informational pages and shared navigation.

## Required Pages

- Home
- About
- Mission
- Timeline
- Publications
- Oral Histories
- Get Involved
- Donate
- Contact
- Privacy Policy

## Required Technical Behavior

- Responsive layout
- Accessible semantic HTML
- Shared header and footer
- Deployment-ready build

## Verification

```bash
npm install
npm run lint
npx tsc --noEmit
npm run build
```
