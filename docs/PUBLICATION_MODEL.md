# Publication Model

## Principle

The publication area functions as a small digital press integrated into the archive.

## Publication modes

The system must support:

- essays
- features
- newsletters
- oral-history narratives
- digital books
- dossiers
- serialized projects
- exhibit texts
- timeline essays
- reference pages
- community voices
- editorial notes

## Contributor system

Contributor records must support:

- name
- biography
- pronouns optional
- portrait optional
- roles
- affiliation optional
- works list
- editorial credits
- exhibit credits

## Editorial workflow

Required states:

- draft
- internal_review
- scheduled
- published
- archived
- withdrawn
