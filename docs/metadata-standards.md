# Metadata Standards

The system combines archival description with digital preservation metadata.

## Archival Description

- DACS
- EAD
- EAC-CPF

## Digital Preservation

- OAIS
- PREMIS
- BagIt
- METS
- Technical metadata (PRONOM, ExifTool, MediaInfo)

## Public Delivery

- Dublin Core
- MODS
- IIIF
- PBCore
- VRA Core
