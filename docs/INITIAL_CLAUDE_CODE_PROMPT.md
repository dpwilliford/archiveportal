# Initial Claude Code Prompt

Build the Sarasota LGBT Archive Public Portal as a custom archival publication CMS using Next.js, TypeScript, Prisma, Tailwind CSS, SQLite for local prototype, PostgreSQL for production, S3-compatible Backblaze B2 media storage, Sharp image derivatives, Matomo analytics, Amazon SES or SMTP email, and Docker deployment.

This is not a preservation repository. It is the public access, publication, exhibit, event, community-submission, and dashboard layer for the Sarasota LGBT Archive. Preservation masters and donor records remain in the separate Archive OS.

The portal must support pages, exhibits, exhibit sections, essays, newsletters, digital books/dossiers, oral-history narratives, timeline entries, collection highlights, images, videos, people, places, contributors, events, volunteer resources, public submissions, correction requests, takedown requests, users, roles, dashboards, audit logs, access logs, and integration status records.

Build a custom CMS, not a generic CMS.

Work in thin vertical slices. Each slice must produce a visible, testable feature.
