# Six-Month Development Roadmap

## Development Assumptions

- Daily development using Claude Code.
- Approximate AI development budget: $200/month.
- Incremental deployment and field testing throughout development.
- GitHub-based source control and documentation.

## Month 1: Foundation

- Establish repository structure.
- Finalize architecture and technical specifications.
- Implement core data models.
- Build application shell.

## Month 2: Intake Workbench Prototype

- Donor intake.
- Rights and consent capture.
- Checksum generation.
- BagIt package creation.

## Month 3: Archive Operating System Core

- SIP ingest.
- Accession assignment.
- PREMIS-style event logging.
- Storage tracking.

## Month 4: Public Portal Prototype

- Collection pages.
- Finding aids.
- Oral history pages.
- Basic search and browse.

## Month 5: Pilot Collections

- Oral history interview.
- Photograph collection.
- Born-digital transfer.
- Digitized newsletter issue.

## Month 6: Stabilization and Launch

- Production deployment.
- Security review.
- Backup and disaster recovery.
- Public launch.

## End-of-Period Capability

By the end of six months, the system should support the complete lifecycle from donor intake and digitization through preservation and public dissemination.
